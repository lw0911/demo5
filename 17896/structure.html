目录与结构说明

站点根目录themes下的文件为可使用的模板文件，您也可以自己建立目录来设计新的模板。

文件结构说明

/themes [模板目录]
 │
 ├─default ------------------[默认模板]
 │   index.html -------------[网站首页]
 │   list.html --------------[文章列表]
 │   content.html -----------[文章详情]
 │   page.html --------------[单页面]
 │   tag_index.html ---------[TAG首页 - TAG合集]
 │   tag_list.html ----------[TAG列表]
 │   tag_content.html -------[TAG内容列表]
 │   search_article.html ----[文章搜索结果]
 │   guestbook.html ---------[留言板 - 自定义表单]
 │   head.html --------------[公共头部]
 │   foot.html --------------[公共底部]
 │   sidebar.html -----------[公共边栏]
 │   error_404.html ---------[404错误页面]
 │
 ├─mobile -------------------[手机模板]
 │   index.html -------------[网站首页]
 │   list.html --------------[文章列表]
 │   content.html -----------[文章详情]
