DuxCms 2.0 官方文档中心

表单内容列表

调用后台自定义表单中的内容，该方法为DuxCms系统模块的方法

使用范围

  • 任何页面

循环调用

<!--循环前缀{app="DuxCms" label="formList" 属性="值" }-->
    ...
<!--{/循环前缀}-->

调用属性

属性 可用值 描述 演示
table 表名 后台添加表单时的表名 table='links'
where sql 扩展SQL条件 无
limit 10 调用数量，默认为10条 limit=10
order data_id ｜自定义字段 排序条件 order="data_id desc"

循环内变量

基本调用格式

{$循环前缀.值}

调用值

属性 描述
data_id 数据 ID
furl 表单内容链接
i 循环序号

其他可用值请请按照后台表单添加后的字段调用，如{$list.name}其中name为添加的字段

示例展示

调用10条友情连接(需要自己添加表单)

<!--list{app="DuxCms" label="formList" table="links" limit=10}-->
  <li><a href="{$list.url}" target="_blank">{$list.name}</a></li>
<!--{/list}-->
