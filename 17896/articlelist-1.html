文章栏目

文章栏目页面模板的可使用标签

使用范围

  • 文章栏目页

页面变量

当前栏目信息

{$categoryInfo.值}

上级栏目信息

{$parentCategoryInfo.值}

顶级栏目信息

{$topCategoryInfo.值}

调用值

属性 描述
class_id 栏目ID
parent_id 上级栏目ID
name 栏目名称
subname 副栏目名称
image 形象图
keywords 关键词
description 描述

分页循环调用

调用栏目下内容信息

<!--foreach{$pageList as $循环前缀}-->
    ...
<!--{/foreach}-->

分页循环内变量

基本调用格式

{$循环前缀.值}

调用值

属性 描述
content_id 内容ID
class_id 栏目ID
title 内容标题
keywords 关键词
description 描述
time 发布时间(格式化后使用)
image 形象图
copyfrom 来源
views 浏览次数
class_name 栏目名称
class_image 栏目形象图
curl 栏目连接
aurl 文章内容连接
content 文章内容
i 循环序号

分页标签

将以下标签放入模板中即可显示出页码链接

{$page}

示例展示

调用当前栏目下的分页内容

<!--foreach{$pageList as $vo}-->
  <li><a href="{$vo.aurl}">{$vo.title}</a></li>
<!--{/foreach}-->

<div class="page"> {$page} </div>
