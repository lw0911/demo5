咨询对应方案

  • 历史悠久，强调精英教育

    美国的中学大多有近百年的发展史，继承了美国传统教育的优良传统，秉承精英教育理念发展至今。

  • 更加多样的课程选择

    美国中学在必修课之外更拥有丰富的选修课系统。课程内容非常丰富，包括视觉艺术、表演艺术、体育、传媒等。

  • 提供语言辅导课程等

    为了帮助国际学生尽快适应纯英文教学环境，大多数的美国中学都开设了ESL课程以供学生学习。ESL课程全名为English as Second Language, 是专门针对母语非英语的国际留学生或新移民而开设的一门课程。主要是关于听、说、读、写方面的训练。

  • 提供升学指导

    美国的学校都有专门的升学指导老师帮助每一个学生更有针对性的选择大学，这些老师大多有着从事招生工作的背景和丰富的指导经验。

  • 小班制教学，注重个性发展

    美国的班级设置与中国有着非常大的差距，每个班最多不超过20人，师生比例也会控制在1：10的范围之内，孩子在这样的班级环境中容易得到更多的关注，拥有更具针对性的培养。

  • 课外活动丰富，注重学生综合素质的培养

    在关注课程设置之外，美国的中学非常注重学生综合素质的培养。他们开设了多样的课外活动以供学生选择。这对学生综合能力的培养是十分有利的。

  • 提供大学预备课程

    为了鼓励学生挑战更高水平的学习，美国的中学开设了荣誉课程，AP课程，AC课程等以供学生选择，这些课程的难度都大于基础课程，学生如果有能力学习这些课程并考核达标后，所修课程在进入大学时可以转换学分，对学生申请优质的大学是有很大帮助的。

  • 学生素质得到全面提高

    美国教育很注重培养学生的全面素质。在中国，学生一旦上了高中，学校、家长和学生的一切中心都是围绕高考。而在美国高中，学校对学生的综合素质培养非常重视，学校的选修课程非常丰富，例如：声乐、器乐等音乐类课程及体育运动类的课程等等，有利于孩子发现、培养自己的兴趣爱好，从而使孩子对学习充满热情，学习积极主动，潜力得到全面挖掘，个性得以充分发挥。
